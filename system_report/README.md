# System Report Script (`system_report.sh`)

## Overview
'system_report.sh` is a lightweight system reporting tool that collects core system metrics and appends them to a persistent log file. It is suitable for routine health checks and hourly scheduled reporting.

---

## Features
- Timestamped system summaries.
- Uptime reporting.
- Disk usage overview .
- Top CPU and memory processes.
- Supports both manual and cron execution.

---

## Requirements
- Bash shell
- Linux utilities: `date`, `uptime`, `df`, `ps`, `head`
- Write access to log directory

---

## Installation and Usage
Place the script in a preferred directory
Edit variables (eg. log destination files ) to preferred locations
